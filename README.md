# Runescape: Dragonwilds — AMP Application Template

This repository provides an AMP (Application Management Panel) template for hosting a **RuneScape: Dragonwilds Dedicated Server** using SteamCMD.  
It enables one‑click installation, automatic updates, config editing, and process management through AMP’s Generic module.

---

## Features

- One‑click install via SteamCMD  
- Automatic updates through AMP  
- Correct dedicated server executable (not the bootstrapper)  
- Exposes the correct game port (UDP 7777)  
- Config editor for `DedicatedServer.ini`  
- Log, save, and config folder shortcuts  
- Clean, community‑friendly structure  

---

## Requirements

- AMP 2.5 or newer  
- Windows host  
- SteamCMD (AMP will handle installation automatically)  
- UDP port **7777** forwarded if hosting publicly
